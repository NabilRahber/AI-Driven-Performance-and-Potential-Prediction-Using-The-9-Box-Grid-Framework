# AI-Driven Performance & Potential Prediction

An ML-based research project using the **9-Box Grid Framework** to categorize employee performance and potential.

## 📊 Key Highlights
* **Dataset:** Analysis of 17,000+ employee records.
* **Models:** Trained using **XGBoost**, **Random Forest**, and **Decision Trees**.
* **Framework:** Mapping results to the HR 9-Box Grid for talent management.
* **Techniques:** Used SMOTE for class balancing and Label Encoding for preprocessing.

## 🛠️ Tech Stack
* Python (Scikit-learn, Pandas, XGBoost)
* Data Visualization (Matplotlib, Seaborn)
* Jupyter Notebook
